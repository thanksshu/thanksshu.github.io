BaiduPCS-Go 简介

发表于 2018-10-20 分类于 实用
本文字数： 1.3k 阅读时长 ≈ 1 分钟

应朋友之请，在此更新关于手机使用BaiduPCS-Go的教程。

2020/03/31 文章已过时

  • 一、需要准备的东西

    1. 一颗耐心，用来仔细阅读博文；
    2. 安装F-droid；
    3. 下载BaiduPCS-Go；
二、准备所需软件

注意！以下命令及输入均以敲击 Enter 做确认！

1.准备 Termux：先安装好 F-droid，并在内部搜索、安装“Termux”；
启动 Termux 并等待初始化，直至初始界面出现；
运行以下代码：

export EDITOR=vi && apt edit-sources

输入“i”来进入编辑模式，
将内部文字替换成：

# The main termux repository:
deb https://mirrors.tuna.tsinghua.edu.cn/termux/ stable main
deb [trusted=yes] http://termux.iikira.com stable main

从屏幕最左侧向右侧滑动，打开面板：

长按“KEYBORD”直到小键盘弹出：

向左滑关闭面板；
按下“ESC”，输入：

:wq

就会保存退出；
接下来输入：

apt update && apt upgrade

更新软件库和包；
最后输入：

termux-setup-storage

并确认，使我们可以访问手机内存，
即可将 Termux 配置完成。
更多使用方法参见：Termux 官网
vi/vim 的使用方法见：菜鸟教程:vi/vim

2.安装BaiduPCS-Go：

apt install BaiduPCS-Go

遇到提示则输入“Y”并 Enter；
等待安装完成……

三、初始化软件

安装好之后先执行：

alias dupan='BaiduPCS-Go'

来简化命令；
再执行：

dupan update

更新到最新版本；
之后是：

dupan config set -appid 266719
dupan config set -savedir /sdcard/BaiduNetdisk
dupan config set -max_parallel=128
dupan config set -cache_size=1024000

分开执行三条指令以初始化客户端；
紧接着：

dupan login

按照提示登录：

输入用户名或其他登录信息；

输入密码；

输入验证码：长按屏幕，弹出如上所示对话框，选择“Select URL”

长按百度验证码的 URL，在浏览器中就可以看到验证码，把它输入进去；

可以说大功告成了。

四、基础使用

这里只介绍最基本的功能，剩余请向我提出更新要求或者看这里

以命令

dupan

进入软件 cli 模式；

此时输入

help

查看命令提示；

执行：

web

在浏览器中访问http://localhost:8080就可以使用（测试功能）了；

使用“CTRL+C”可以退出程序，
或者在 cli 模式下输入

quit

也可以；

在 cli 模式下（“#”后是注释）：

cd 路径名 #进入指定目录
ls #查看路径下的文件
pwd #显示当前路径
quota #查看网盘容量信息
d 文件名或目录名 #下载文件，默认为本地目录下的BaiduNetdisk内

……
